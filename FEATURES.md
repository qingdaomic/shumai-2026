## 一、产品定位

**一句话**：中考数学 AI 私人教练

**核心理念**：做题只做减法，冲满分。不做题海战术，1334 道精选题，题题经典。

**目标用户**：初中 7-9 年级学生（主），家长（辅），教师（扩展）

**差异化**：
- 不是"题库壳"，是"带方法论的学习系统"
- 不是"刷题机器"，是"知道你哪里不会、为什么不会、怎么补"
- AI 不是噱头，是每个知识点、每道题都能一键讲解的真实能力

## 二、核心功能矩阵

### 2.1 知识体系（学什么）

| 功能 | 描述 | 亮点 | 宣传角度 |
|------|------|------|----------|
| **194 知识点技能树** | 覆盖初中三年完整数学体系，按 2025 北师大版新课标编排 | 不是随意罗列，是按新课标+考频排序的结构化知识树 | "194个知识点，一个不漏，按考频排优先级" |
| **知识图谱可视化** | 24 个核心节点的依赖关系图，点击跳转详情 | 让学生"看见"知识之间的联系 | "数学不是一盘散沙，看见知识的脉络" |
| **三大领域** | 数与代数(90) + 图形与几何(82) + 统计与概率(22) | 领域进度条直观显示掌握程度 | "三大领域，一目了然你的强弱" |
| **每日晨读卡** | 每个知识点配口诀 + 易错提醒 | 轻量记忆，不占时间 | "每天2分钟，口诀记牢不丢分" |

### 2.2 题库体系（练什么）

| 功能 | 描述 | 亮点 | 宣传角度 |
|------|------|------|----------|
| **624 道中考真题** | 10 个城市（青岛/北京/上海/广州等）10年真题 | 按城市、年份、难度、知识点多维筛选 | "10年10城真题，考过的题最有价值" |
| **500 道基础习题** | 每个知识点配套基础练习，diff 1-2 | 做错自动降级补差，做对自动升级 | "基础不牢地动山摇，500题夯实根基" |
| **35 组题组训练** | 按解题方法分组，每组 4-6 题 | 不是随机凑，是"同方法不同题"的刻意练习 | "同一种方法练6遍，形成条件反射" |
| **20 组压轴题组** | diff 4-5 综合题，分步拆解 | 每题有分步策略，不是直接给答案 | "压轴题不可怕，分步拿分有策略" |
| **23 种解题方法** | 配方法/换元法/面积法/构造法等 | 方法与知识点双向映射 | "不只教你答案，教你方法" |
| **方法卡收集** | 集齐 23 张方法卡 = "方法大师"成就 | 游戏化激励 | "集齐23张方法卡，你就是方法大师" |

### 2.3 智能诊断（知道哪不会）

| 功能 | 描述 | 亮点 | 宣传角度 |
|------|------|------|----------|
| **首次诊断** | 10 道选择题，5 分钟，AI 定位起点 | 不浪费时间从头学，直接找到你的起点 | "5分钟找到你的数学起点" |
| **BFS 错题追因** | 做错一道题 → 沿知识依赖树向下追溯根因 | 不止告诉你"错了"，告诉你"为什么错" | "错在表面，根在深处，帮你追到底" |
| **薄弱知识点 TOP3** | 动态计算未掌握+高考频的知识点 | 实时更新，永远知道该补什么 | "永远知道下一步该学什么" |
| **能力雷达图** | 6 维能力可视化（代数/几何/统计/函数/方法/速度） | 一张图看清全面能力 | "六维雷达，一眼看清你的数学能力" |
| **分数预测** | 基于掌握度 × 考频权重 → 预测中考分数 | 非线性映射，"你在这里"标注 | "你现在能考多少分？AI 告诉你" |
| **中考预测** | 基于 10 年真题数据分析命题趋势 | 必考点 + 高概率点 + 复习优先级 | "AI 分析10年真题，预测今年考什么" |

### 2.5 试卷分析（逆向学习）

| **试卷逆向拆解** | 粘贴一套试卷 → AI 逐题分析知识点+难度+考点 | 正向学习（知识→题）+ 逆向分析（试卷→知识点→补漏） | "一套试卷，拆出你的满分路径" |

## 三、技术亮点（面向技术合作/融资）

| 技术 | 说明 |
|------|------|
| **知识图谱 + BFS 追因** | 不是简单的对错统计，是基于图论的知识依赖分析，错题追溯到根因 |
| **双 AI 模型** | DeepSeek + 豆包，支持模型切换，不被单一供应商绑定 |
| **结构化 Prompt 体系** | 6 种场景化 Prompt（讲解/错题分析/知识总结/语音脚本/对话/学习路径） |
| **遗忘曲线调度** | 基于 Ebbinghaus 遗忘曲线的间隔复习算法（1天→3天→7天→30天） |
| **单文件到多文件架构演进** | 从 7000 行单文件原型 → Vite 工程化 → 前后端分离，保持产品连续性 |
| **三端响应式** | 手机(底部Tab) / Pad(图标侧边栏) / 桌面(完整侧边栏)，一套代码三种体验 |
| **PWA + 离线支持** | Service Worker 缓存策略，弱网/离线可用 |
| **微信 ClawBot** | 基于腾讯官方 iLink 协议，合法不封号 |
