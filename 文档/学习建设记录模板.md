# 树脉学习建设记录模板

> 用途：给树脉每个阶段的建设记录统一写法，保留日记式文风、截图证据和自媒体复用出口。

## 统一写法

每个阶段建议都按下面顺序写：

```md
### [YYYY-MM-DD HH:mm] 阶段名称

- 行动：
- 目的：
- 结果：
- 困难：
- 解决：
- 截图：
- 下一步：

当下感受：

写一两段自然一点的话，保留判断、犹豫、发现和完成后的轻松感。

可复用摘要：

- 一句话结果
- 一句产品判断
- 一句下一步
```

## 写作建议

- 用短句，不要写成机器流水账。
- 如果有界面变化，优先把截图路径写清楚，再在主建设记录里引用。
- 如果这一步适合整理成小红书、抖音或公众号，最后补 3 条可复用摘要。
- 如果是纯文档或纯后端阶段，也保留“当下感受”，让记录像人在写。

## 复用方式

- 新阶段开始时，直接复制这个模板。
- 完成后，再把“行动 / 结果 / 下一步”压缩成主建设记录里的日常段落。
