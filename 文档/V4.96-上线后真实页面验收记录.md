真实浏览器验收时，后台页面能打开，但后台 Tab 中没有 `Skill 管理`。

排查后确认：

- GitHub HEAD 的 `src/App.jsx` 已包含 `Skill 管理`
- release `dist/assets/index-BBjZDaHT.js` 已包含 `Skill 管理` 与 `operation-logs`
- 但公网 `https://shumai.cc/` 当时仍加载旧 asset：`/assets/index-BqO7Cdc5.js`

进一步只读检查 Nginx 配置发现：

- Nginx 实际 root：`/opt/shumai/dist`
- V4.94 操作卡发布到：`/var/www/shumai`

所以 V4.94 后端已正式切换成功，但前端公网仍在由 `/opt/shumai/dist` 的旧构建承载。
