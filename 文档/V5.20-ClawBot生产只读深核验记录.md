## 验证结果

### 公网 API

- `GET https://shumai.cc/api/health`：200，后端正常
- `GET https://shumai.cc/api/wechat/status`：401 `未登录`
- `POST https://shumai.cc/api/wechat/unbind`：401 `未登录`
- `GET https://shumai.cc/api/daily`：401 `未登录`
- `GET https://shumai.cc/api/parent/children`：401 `未登录`
- `GET https://shumai.cc/api/study-plan`：404

结论：`wechat`、`daily`、`parent` 三组接口已在公网后端挂载并受登录鉴权保护；`study-plan` API 仍未在当前生产后端挂载。

### PM2 / Nginx / release

- PM2 当前只有一个线上进程：`shumai-api-v4.94:online`
- PM2 进程用户：`root`
- PM2 cwd：`/opt/shumai-releases/v4.94-20260509-190052/repo/server`
- PM2 exec：`/opt/shumai-releases/v4.94-20260509-190052/repo/server/index.js`
- 未发现正在运行的 `bot.js` / claw / wechat / openclaw 相关进程
- Nginx 当前正式前端 root：`/opt/shumai/dist`
- Nginx `/api/*` proxy：`http://localhost:3001`

结论：生产后端仍是 V4.94 release；ClawBot 独立 bot 进程当前未在线。
