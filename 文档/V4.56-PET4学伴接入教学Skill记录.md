# V4.56 PET-4 学伴接入教学 Skill 记录

最后更新：2026-05-09

## 目标

让树脉学伴从“状态陪伴”继续向“行动陪伴”前进：学生卡住时，不必先想怎么提问，可以直接点学伴给出的学习动作。

## 本阶段完成

- 新增 `PET_SKILL_ACTIONS`
- 新增 `triggerLearningPetSkillAction()`
- 新增 `shumai-pet-skill-action` 本地事件通道
- 学伴面板新增三个动作按钮：
  - 给我一个突破口
  - 判断错因
  - 来一道更简单的
- 全局 AI 浮窗 `AIFloat` 监听学伴动作：
  - 自动打开浮窗
  - 自动填入对应提问词
  - 如果当前在知识点详情页，会带入当前知识点上下文

## 设计判断

本阶段不直接让学伴自动发送 AI 请求，而是先把提问词填到 AI 浮窗输入框里，让学生保留最后一步确认权。

这样既能降低提问门槛，也避免学伴过度打断学生。它像一个站在旁边的小教练，递出三个清楚的下一步，而不是抢走题目现场。

## 边界

- 不接新后端
- 不改数据库
- 不执行 SQL
- 不部署
- 不 SSH
- 不 PM2 / Nginx
- 不触碰禁止文件
