# V4.54 PET-2 有效学习行为 XP 记录

最后更新：2026-05-09

## 目标

让树脉学伴的成长绑定有效学习行为，而不是绑定在线时长、刷新或反复点击。

## 本阶段完成

- 新增 `awardLearningPetXp()` 本地奖励通道
- 学伴监听 `shumai-pet-xp` 事件并更新 XP
- 奖励记录使用 `localStorage` 去重，避免同一动作反复刷 XP
- 学伴面板显示最近一次奖励

## 已接入行为

- 掌握一个知识点：`+30 XP`
- 完成一道真题错题订正：`+15 XP`
- 修复一道基础错题：`+15 XP`
- 完成今日晨读：`+20 XP`

## 设计判断

第一版只奖励已经有清晰状态更新的动作，不奖励页面停留、打开解析、切换学伴状态或反复点击。

“做对一道新题”的独立奖励暂不接，因为当前题目页还没有统一的“独立做对”判定入口，后续可在作答区统一改造时接入。

## 边界

- 不接后端
- 不新增数据库表
- 不执行 SQL
- 不部署
- 不 SSH
- 不 PM2 / Nginx
- 不接微信
