# V4.53 PET-1 树脉学伴前端第一版记录

最后更新：2026-05-09

## 目标

做出第一版网页内“树脉学伴”，让它成为学生学习成长的可视化分身，而不是普通装饰。

## 本阶段完成

- 新增 `LearningPet`
- 新增 `PetSprite`
- 支持右下角展开 / 收起
- 支持小猫 / 小狗切换
- 支持 5 种状态：
  - `idle`
  - `thinking`
  - `waiting`
  - `review`
  - `happy`
- 显示 XP、等级和成长阶段
- 使用 `localStorage` 保存状态
- 桌面默认展开，手机默认收起成入口
- 手机端放在底部导航上方，减少对题目阅读和作答按钮的遮挡

## 成长口径

第一版先只做状态和成长展示，不接真实学习行为加 XP。

真实 XP 增长放到 PET-2：

- 做对题
- 完成订正
- 掌握知识点
- 完成今日任务

## 设计判断

学伴使用克制的卡片和轻量图形表达，不做复杂游戏化，也不制造焦虑。它的第一目标是陪伴学生回到下一步学习动作。

## 边界

- 不接后端
- 不新增数据库表
- 不部署
- 不 SSH
- 不 PM2 / Nginx
- 不接微信
- 不接商城 / 装扮 / 上传图片
