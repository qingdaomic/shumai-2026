# V4.67 SKE/PET 推荐调整建议记录

日期：2026-05-09

## 目标

把后台“低效学伴动作观察清单”推进为可行动建议。

V4.65 能看到哪些 Skill 需要观察，V4.66 能看到问题主要发生在哪个入口。V4.67 不做自动降权，而是先给管理员一个克制、可解释的建议标签。

## 本阶段完成

- 观察清单新增 `pet_recommendation`
- 观察清单新增 `pet_recommendation_reason`
- 建议类型包括：
  - `lower_weight`：建议降权
  - `rewrite_prompt`：建议改写
  - `adjust_entry`：调整入口
  - `keep_observing`：保留观察
- 后台卡片展示建议标签和原因
- 建议仍然只展示，不自动修改权重、不自动禁用 Skill

## 建议规则

- 没帮助反馈连续偏高，且高于有帮助：建议降权
- 已有没帮助反馈：建议改写提示词
- 多次触发但未进入 AI 使用：建议调整入口
- 样本仍少：保留观察

## 边界

- 不自动改权重
- 不自动禁用 Skill
- 不新增数据库表
- 不执行 SQL
- 不新增后端接口
- 不部署
- 不服务器操作
- 不 SSH
- 不 PM2 / Nginx
- 不触碰 TTS / ASR / study-plan / deploy / schema / public animations 等禁止或历史脏改文件
