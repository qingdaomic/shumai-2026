# V4.62 SKE/PET 后台 Skill 来源筛选记录

最后更新：2026-05-09

## 目标

让后台 Skill 管理页不只是看到来源统计，还能按来源筛选 Skill，便于单独观察学伴动作、斜杠提问、搜索等入口的实际效果。

## 本阶段完成

- `GET /api/admin/skills` 新增 `source` 查询参数
- 支持来源：
  - `slash_prompt`
  - `skill_answer`
  - `learning_pet`
  - `search`
  - `unknown`
- 后台 Skill 管理页新增“全部来源”筛选下拉
- 管理员可单独筛出学伴来源有事件的 Skill

## 设计判断

来源筛选不新增数据库字段，继续复用 `prompt_skill_events.meta.entry` 的聚合结果。后端只允许白名单来源，避免把任意查询值拼进 SQL。

## 边界

- 不新增数据库表
- 不执行 SQL
- 不部署
- 不 SSH
- 不 PM2 / Nginx
- 不触碰 TTS / ASR / study-plan / deploy / schema 等禁止或历史脏改文件

## 后续

- 根据来源筛选结果观察学伴动作质量
- 学伴动作根据 `helpful / not_helpful` 做排序
- 高效动作沉淀为题目页默认建议
