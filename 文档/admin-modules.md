# 中考数学系统管理端模块设计文档

> 版本：V1.0 | 对应阶段：V1.5 基础管理端 → V2 完整运营后台

---

## 一、管理端定位

管理端是平台的"内容与运营控制台"，不是学员使用的前台。

主要用户：

- **管理员（admin）**：全部功能，含财务和系统配置
- **教研/运营（operator）**：内容维护、视频审核，无财务权限

管理端建议独立部署（独立子域名或路径），与学员端完全分离。

## 三、各模块详细设计

---

### 3.1 登录页 `/admin/login`

**页面字段**：

- 手机号输入框
- 密码输入框
- 登录按钮

**逻辑**：

- 登录成功后跳转仪表盘
- role = student 的账号登录后拒绝，提示"无管理权限"
- 支持记住登录状态（7 天 token）

---

### 3.2 仪表盘 `/admin/dashboard`

**页面区域**：

#### 数据概览卡片（今日 / 累计）

| 指标 | 说明 |
|------|------|
| 总学员数 | 所有注册用户数 |
| 今日活跃 | 今日有操作的学员数 |
| 今日新增 | 今日注册数 |
| 付费学员数 | status = active 总数 |
| 今日做题数 | 所有学员今日提交的做题记录数 |
| 今日视频点击 | 今日视频点击总次数 |
| 错题本总条目 | 全平台 wrong_book_records 总数 |
| 诊断总次数 | 全平台 diagnosis_records 总数 |

#### 快捷入口

- 待审核视频数量（点击跳转视频候选池）
- 失效视频数量（点击跳转失效报告）
- 近 7 天新注册趋势图

---

### 3.3 学员管理 `/admin/students`

#### 学员列表页

**筛选条件**：

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 关键词 | 文本 | 按手机号/昵称模糊搜索 |
| 状态 | 选择 | active / trial / expired / inactive / banned |
| 套餐 | 选择 | 按 plan_id 筛选 |
| 注册时间 | 日期范围 | — |
| 套餐到期时间 | 日期范围 | — |

**列表字段**：

| 字段 | 说明 |
|------|------|
| ID | — |
| 手机号（脱敏） | — |
| 昵称 | — |
| 状态 | 彩色标签 |
| 套餐 | 套餐名称 |
| 套餐到期 | 日期 |
| 注册时间 | — |
| 累计学习时长 | 分钟数 |
| 操作 | 查看详情、修改状态、赋予套餐 |

**操作**：

- 查看学员详情
- 修改学员状态（active / banned / inactive）
- 手动赋予套餐（指定套餐 + 有效天数）
- 导出 CSV

#### 学员详情页

**信息区域**：

- 基础信息：手机号、昵称、状态、注册时间、最后登录时间
- 档案信息：年级、学校、目标分数、考试城市
- 套餐信息：当前套餐名、到期时间、历史订单
- 学习统计：做题数、正确率、错题数、诊断次数、学习时长
- 知识点掌握概览：各知识点 mastery_level 列表
- 最近错题列表（最近 10 条）
- 最近诊断记录（最近 3 次）
- 操作日志（最近 20 条行为记录）

### 3.4 知识点管理 `/admin/topics`

#### 知识点列表页

**筛选**：grade / category / status / keyword

**列表字段**：

| 字段 | 说明 |
|------|------|
| code | 业务 ID |
| name | 知识点名称 |
| grade | 年级阶段 |
| category | 类别 |
| difficulty | 难度星级 |
| 视频数 | 关联已发布视频数量 |
| 状态 | draft / published / archived |
| 操作 | 编辑、发布/下线 |

#### 知识点编辑页

**字段**：

- code（编辑时不可修改）
- name
- grade（选择）
- category（选择）
- difficulty（1-5 滑动选择）
- definition（富文本或 Markdown 编辑器）
- formula（支持数学公式输入）
- trap（易错点）
- slogan（晨读口诀）
- sort_order

**底部关联预览**：

- 本知识点已关联题目数量
- 本知识点已关联视频列表
- 前置/后置知识点依赖关系

### 3.7 视频资源中心 `/admin/videos`

这是优先级最高的管理端模块，详细设计参见 `video-resource-design.md`。

#### 3.7.1 视频列表（含状态 Tab 切换）

Tab 分类：全部 / 待审核 / 已发布 / 已拒绝 / 已下线 / 链接失效

**列表字段**：

| 字段 | 说明 |
|------|------|
| 封面/标题 | — |
| UP主 | — |
| 时长 | — |
| 适用层次 | basic / intermediate / advanced |
| 质量分 | 审核打分 |
| 关联知识点 | 已关联的知识点列表（最多显示 3 个） |
| 点击量 | 已发布视频的总点击数 |
| 状态 | 彩色标签 |
| 链接状态 | 正常 / 失效 |
| 最近检测 | last_checked_at |
| 操作 | 审核、关联知识点、下线、删除 |

#### 3.7.2 视频审核页

**展示**：

- 视频标题、URL（点击跳转 B 站预览）
- UP 主、时长、简介
- 提交人、提交时间

**操作**：

- 质量分滑块（1-100）
- 适用层次选择
- 拒绝原因输入框（拒绝时必填）
- "通过审核" / "拒绝" 按钮

#### 3.7.3 知识点关联配置页

**展示当前知识点的所有视频**（含主视频和备选）：

- 拖拽调整优先级顺序
- 切换主视频
- 单条隐藏或移除关联
- 新增关联（从已审核视频中搜索选择）

#### 3.7.4 链接检测报告页

**展示**：

- 失效视频列表（is_available = 0）
- 最近检测时间
- 连续失效次数
- 受影响的知识点
- 操作：手动重新检测 / 标记为下线 / 替换为其他视频

#### 3.7.5 视频录入页

**字段**：

- 平台（默认 bilibili）
- B 站视频 URL（输入后自动解析 BV 号）
- 标题
- UP 主
- 时长（手动填写或自动解析）
- 适用层次
- 年级范围
- 标签（多选）
- 备注

**提交后**：进入 `pending` 候选池，等待审核。

### 3.11 系统配置 `/admin/settings`

#### 3.11.1 功能开关

展示所有 `feature_flags` 条目，支持一键开启/关闭。

常用开关示例：

| key | 说明 |
|-----|------|
| enable_video | 是否展示视频讲解入口 |
| enable_ai_diagnosis | 是否启用 AI 诊断建议 |
| enable_registration | 是否开放新用户注册 |
| maintenance_mode | 维护模式（前台展示维护提示） |
| trial_basic_limit | 试用期基础题每知识点限制数量 |

#### 3.11.2 公告管理

- 公告列表
- 新建/编辑公告
- 设置展示时间段
- 设置目标角色（全部学员 / 试用 / 付费）
- 置顶开关

#### 3.11.3 管理员账户

- 查看所有 admin / operator 账户
- 新建管理员账户（只能在此处创建，不对外开放注册）
- 停用/恢复管理员账户
- 修改管理员角色

## 四、管理端通用 UI 规范

### 4.1 布局

- 左侧固定导航栏
- 顶部面包屑 + 用户信息
- 主内容区

### 4.2 列表页规范

- 统一筛选条在顶部
- 表格支持分页（每页 20 / 50 / 100 条可选）
- 操作列固定在右侧
- 状态字段统一用彩色标签

### 4.3 编辑页规范

- 表单字段标注必填/选填
- 保存前有基础校验
- 修改未保存时离开给出提示
- 保存成功/失败有明确提示

### 4.4 危险操作规范

以下操作必须有二次确认弹窗：

- 封禁学员
- 拒绝视频（需填写原因）
- 下线内容（题目/知识点/视频）
- 退款处理
- 功能开关的开启/关闭

## 五、V1.5 管理端最小实现范围

V1.5 轻后端阶段，管理端只需做以下模块：

1. **登录页**
2. **仪表盘**（基础指标卡片）
3. **学员管理**（列表 + 详情 + 修改状态）
4. **视频资源中心**（录入 + 审核 + 知识点关联）
5. **数据统计**（简版总览）

其他模块在 V2 阶段再补。
