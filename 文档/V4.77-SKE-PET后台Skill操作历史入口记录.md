# V4.77 SKE/PET 后台 Skill 操作历史入口记录

日期：2026-05-09

## 目标

V4.76 已经在 Skill 行内显示最近一次人工操作。V4.77 继续补一个轻量入口：管理员可以从某个 Skill 行直接打开它的操作历史，查看最近降权或改写记录。

## 本阶段完成

- Skill 行操作区新增“历史”按钮
- 点击后打开 Skill 操作历史面板
- 优先读取 `GET /api/admin/operation-logs?targetType=prompt_skill&targetKey=...`
- 最多读取 20 条
- API 不可用或返回 `stored:false` 时显示本地降级记录
- 历史面板支持刷新和收起
- 历史面板显示持久化 / 本地降级 / 读取中状态
