# 树脉截图记录

本目录用于保存树脉系统每个阶段性界面设计的截图。

命名建议：

- `YYYY-MM-DD-v阶段-页面-视口.png`
- 示例：`2026-05-08-v4-16-auth-register-mobile.png`
- 示例：`2026-05-08-v4-16-auth-scan-desktop.png`

使用规则：

- 每个阶段完成 UI / 页面 / 交互调整后，至少保存一张关键截图。
- 重要页面建议同时保存桌面端和移动端截图。
- 截图文件放入本目录，避免散落在 `/private/tmp`。
- `文档/树脉系统建设记录.md` 中必须用 Markdown 图片引用对应截图。
- 不把图片二进制或 Base64 直接写进 `.md` 文件；统一采用“图片文件 + Markdown 相对路径引用”。
- 截图引用使用相对路径，例如：

```md
![V4.16 注册页移动端](截图记录/2026-05-08-v4-16-auth-register-mobile.png)
```
