# V4.58 PET-6 成长图鉴与家长端记录

最后更新：2026-05-09

## 目标

让学伴成长被学生和家长看见，但不把它做成排名、攀比或焦虑面板。

## 本阶段完成

- 新增 `PET_ACTIVITY_LOG_KEY`
- XP 奖励时同步写入本地成长活动日志
- 学伴面板新增“成长图鉴”
  - 当前成长徽章
  - 本周 XP
  - 本周掌握知识点数量
  - 本周修复错因数量
  - 最近三条有效学习记录
- 家长端新增“学伴成长信号”
  - 新掌握
  - 修复错因
  - 坚持痕迹
- 家长端文案强调“看见努力，不做排名”

## 设计判断

学伴成长不是游戏排行榜，而是把有效学习行为翻译成孩子能理解、家长能鼓励的证据。

学生端看到的是：我今天真的变强了一点。  
家长端看到的是：孩子哪里有努力，今晚该肯定哪一个具体动作。

## 边界

- 不新增后端接口
- 不改数据库
- 不执行 SQL
- 不部署
- 不 SSH
- 不 PM2 / Nginx
- 不触碰服务器和禁止文件
