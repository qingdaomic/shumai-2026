# V4.69 SKE/PET 建议改写入口草案记录

日期：2026-05-09

## 目标

把“建议改写”的学伴 Skill 从只读建议，推进到可人工编辑的后台入口。

这一阶段仍然不让系统自动改写提示词。管理员必须在观察清单里点击“人工改写”，检查内容后手动保存。

## 本阶段完成

- 扩展已有 `PUT /api/admin/skills/:id`，支持保存 `content`
- 不新增接口
- 学伴低效观察卡片中，`pet_recommendation = rewrite_prompt` 时显示“人工改写”
- 点击后打开“人工改写 Skill”编辑面板
- 保存前校验内容不能为空
- 保存后刷新 Skill 列表和观察清单
