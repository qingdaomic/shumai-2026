# 数脉题库规划表 v1.0

> **核心理念：1500 道题，题题经典，不做题海战术。**
>
> 每道题必须满足：① 有完整分步解题过程 ② 几何题有 SVG 图形 ③ 有明确的考查意图 ④ 有对应知识点和方法标签

## 一、总体架构

| 模块 | 定位 | 题数 | 难度 | 目标 |
|------|------|------|------|------|
| **基础习题** | 概念理解 + 基本计算 | ~550 道 | diff 1-2 | 每个知识点夯实基础，0 失误 |
| **题组训练** | 方法专练 + 模式识别 | ~350 道（58 组×6 道） | diff 2-3 | 按解题方法分类，练出条件反射 |
| **压轴突破** | 综合应用 + 分步突破 | ~300 道 | diff 4-5 | 拿满分的关键，每题有分步拆解 |
| **综合模拟** | 考场仿真 + 查漏补缺 | ~300 道（25 套×12 道） | diff 1-5 混合 | 限时训练，找薄弱环节 |
| **合计** | | **~1500 道** | | |

> 另有 **中考真题** 624 道（EXAM_QS，id:0-623，独立模块，不计入 1500）
> 当前系统已有题目：624(真题) + 500(基础) + 137(题组) + 58(压轴) + 15(诊断) = **1334 道**（截至 2026-04-27）

## 二、题数分配原则

### 权重公式

```
每个知识点的题数权重 = 考频(freq) × 难度系数(1 + diff/5)
```

- 考频 ≥ 90 的知识点：**一线考点**，题量最多
- 考频 70-89：**重点考点**，中等题量
- 考频 50-69：**基础考点**，保底题量
- 难度越高的知识点，需要更多梯度练习

### 每道题的质量标准

| 要求 | 说明 |
|------|------|
| **sol 字段** | 分步解答，格式：`步骤1：…\n步骤2：…\n答：…` |
| **svg 字段** | 几何题必须有内联 SVG（统一风格） |
| **methods** | 标注本题用到的解题方法编号 |
| **subTopics** | 标注涉及的所有知识点 |
| **error 字段** | 标注常见错误（如有） |
| **difficulty ladder** | 同一知识点内，题目难度从 1→2→3 递进 |

## 五、分批执行计划

### 第 1 批：高频基础（目标 ~350 道，2-3 周）

**优先级**：考频 ≥ 85 的一线考点基础题 + 部分题组

覆盖知识点：
- 代数：rational, linear_eq, eq_app, linear_fn, quad_eq, quad_fn, quad_fn_app
- 几何：congruent, pythagorean, parallel_lines, similar, circle_angle, circle_tangent
- 统计：stats, mean_median, prob

交付标准：每题有 sol（分步骤）+ 几何题有 svg

### 第 2 批：重点题组（目标 ~400 道，3-4 周）

**优先级**：23 种解题方法的对应题组 + 中等考频知识点基础题

覆盖知识点：
- 补齐所有考频 70-84 知识点的基础题
- 完成全部 58 组题组训练

### 第 3 批：压轴突破（目标 ~300 道，3-4 周）

**优先级**：diff 4-5 的综合题 + 分步解题

重点：
- 二次函数压轴（12 道梯度题）
- 相似+圆综合（8 道）
- 旋转+坐标综合（4 道）
- 全部压轴题必须有分步拆解（步骤 1→步骤 2→...）

### 第 4 批：模拟套卷 + 查漏补缺（目标 ~350 道，2-3 周）

- 25 套模拟卷
- 补齐低频知识点
- 全套最终验证
